HTML attributes

  1. alt
    2. Specifies an alternative text for an image, when the image cannot be displayed

  2. disabled
    3. Specifies that an input element should be disabled

  3. href
    4. Specifies the URL (web address) for a link

  4. id
    5. specifies a unique id for an element

  5. src
    6. Specifies the URL (web address) for an image

  6. style
    7. Specifies an inline CSS style for an element

  7. title
    8. specifies extra information about an element (displayed as a tool tip)
